# update-catchphrase

## Summary
サービスの説明文（キャッチフレーズ）を、より魅力的で印象的な表現に更新する。新しいフレーズは「映画館のような没入感で、まだ見ぬ傑作と出会おう。」とし、サービスのシネマティックな体験と、新しい映画との出会いという価値をより明確に伝える。

## Motivation
現在の説明文「次に観る映画をシネマティックに発見しよう。」は機能的だが、ユーザーに対してサービスの魅力や体験価値を十分に伝えきれていない。特に以下の点を改善したい：

- **没入感の強調**: 映画館のような没入体験という独自の価値をより明確に
- **出会いの演出**: 未知の傑作との出会いというエモーショナルな訴求
- **記憶に残る表現**: より印象的で記憶に残りやすいフレーズに

新しいキャッチフレーズは、以下の箇所で使用される：
- meta descriptionタグ（SEO、検索結果での表示）
- OGPメタタグ（SNSでシェアされた際の表示）
- Twitter Cardメタタグ（Twitterでの表示）
- スタートモーダルのウェルカムメッセージ
- 情報（About）モーダル内のアプリ説明

## Scope
- **変更対象**: `client/index.html` 内の複数箇所
- **影響範囲**: 軽微（テキスト変更のみ、ロジックや動作への影響なし）
- **既存機能**: 影響なし

## Impact
### User Impact
- ユーザーがサービスの価値をより明確に理解できる
- SNSシェアや検索結果でのクリック率向上が期待できる
- 初回訪問時のウェルカムメッセージがより魅力的に

### Technical Impact
- HTMLファイルのテキスト変更のみ
- 後方互換性: 完全に維持（動作変更なし）
- パフォーマンス: 影響なし

## Dependencies
なし

## Alternatives Considered
以下の代替案も検討したが、最終的に現在の案を採用：

1. 「予告編をザッピング、運命の1本と出会おう。」
   - サービス名の由来を活かした案だが、「ザッピング」という言葉が若干古い印象

2. 「次々と予告編を巡って、新たな作品との出会いを。」
   - 素直な表現だが、インパクトに欠ける

3. 「予告編が導く、あなただけの映画体験。」
   - パーソナライズを強調した案だが、実際の機能（配信サービスフィルター等）との整合性がやや弱い
