# PWA Icons

このディレクトリには Progressive Web App (PWA) 用のアイコンファイルが含まれます。

## 必要なアイコン

- **icon-192x192.png**: 192x192ピクセルのPNG画像
- **icon-512x512.png**: 512x512ピクセルのPNG画像

## アイコンデザインガイドライン

- 映画フィルムや予告編を連想させるデザイン
- maskable アイコン対応（Safe Area: 中心の80%にメインコンテンツを配置）
- 背景色: #050914 （アプリのテーマカラー）
- ファイルサイズ: 192x192は50KB以下、512x512は200KB以下を推奨

## 現在の状態

現在のアイコンファイルは一時的なプレースホルダーです（favicon.icoのコピー）。
適切なPNGアイコンを作成して置き換えてください。
