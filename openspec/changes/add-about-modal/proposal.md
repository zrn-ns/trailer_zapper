# Proposal: 情報モーダルの追加

## 概要

アプリケーション情報（ライセンス、作者情報、ソースコード、TMDB表記）を表示する専用のモーダルページを追加します。

## 背景

現在、TMDBとOtoLogicのライセンス表記をUI下部に配置していますが、画面下部で見切れてしまう問題があります。また、作者情報やソースコードへのリンクなど、その他の情報も一箇所にまとめて表示する必要があります。

## 目標

- ライセンス表示が見切れる問題を解決
- すべてのアプリケーション情報を一箇所に集約
- ユーザーがいつでもアクセスできる情報モーダルを提供

## 実装内容

1. **情報ボタンの追加**: UI上部に「情報」ボタンを配置
2. **情報モーダルの実装**: 以下の情報を含むモーダル
   - TMDBのロゴとデータ提供表記
   - OtoLogic音声ライセンス (CC BY 4.0)
   - 作者情報
   - ソースコードへのリンク（GitHubなど）
3. **既存表記の削除**: 現在のUI下部の`.tmdb-attribution`と`.audio-attribution`を削除

## 影響範囲

- `client/index.html`: モーダルHTML、情報ボタンの追加、既存表記の削除
- `client/script.js`: モーダル開閉ロジックの追加
- `client/style.css`: モーダルスタイル、既存表記スタイルの削除

## 備考

- 既存のジャンルフィルターモーダルの実装パターンを参考にする
- モーダルはバックドロップクリックまたは閉じるボタンで閉じられる
