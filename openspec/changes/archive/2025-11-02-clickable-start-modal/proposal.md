# Proposal: Clickable Start Modal

## Overview

上映開始画面（start-modal）において、「上映開始」ボタン以外の画面全体をクリックしても上映を開始できるようにする。これにより、ユーザーが画面のどこをクリックしても直感的に上映を開始できるようになり、UXが向上する。

## Motivation

現在の実装では、上映開始画面で「上映開始」ボタンのみがクリック可能です。しかし、映画館での体験をシミュレートする本アプリケーションにおいて、画面全体をクリックして開始できるようにすることで、より直感的な操作が可能になります。

## Scope

- UI/UX: 上映開始画面の操作性改善
- 関連コンポーネント: start-modal、イベントハンドラー

## Alternatives Considered

1. **現状維持**: ボタンのみをクリック可能にする
   - 採用しない理由: ユーザーが意図せず空白部分をクリックした場合、何も起きないため混乱を招く可能性がある

2. **ボタンを大きくする**: ボタンのサイズを大きくして、クリック可能領域を広げる
   - 採用しない理由: デザインのバランスが崩れる可能性がある

3. **画面全体をクリック可能にする（採用案）**: 上映開始画面全体をクリックして上映を開始できるようにする
   - 採用理由: 最も直感的で、既存のデザインを維持しながらUXを改善できる

## Dependencies

なし

## Timeline

小規模な変更のため、即座に実装可能。

## Success Criteria

- [x] 上映開始画面全体（`#start-modal`）をクリックして上映を開始できる
- [x] 既存のボタンクリック機能も引き続き動作する
- [x] キーボードショートカット（Space, Enter）も引き続き動作する
- [x] 重複クリックを防止して、処理が1回のみ実行される
